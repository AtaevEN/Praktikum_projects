# Статистический анализ данных сервиса аренды самокатов GoFast

### [Вернуться обратно к проектам](https://github.com/AtaevEN/Praktikum_projects/tree/main)

## Описание проекта  
Вы аналитик популярного сервиса аренды самокатов GoFast. В вашем распоряжении данные о некоторых пользователях из нескольких городов, а также информация об их поездках.

Пользователи совершают поездки с помощью мобильного приложения, которое предлагает два тарифных плана:  
- **Без подписки**  
  - Абонентская плата отсутствует  
  - Стоимость одной минуты поездки — 8 рублей  
  - Стоимость старта поездки — 50 рублей  
- **С подпиской Ultra**  
  - Абонентская плата — 199 рублей в месяц  
  - Стоимость одной минуты поездки — 6 рублей  
  - Стоимость старта — бесплатно  

Основная задача — проанализировать данные и проверить гипотезы, которые помогут бизнесу улучшить сервис и увеличить прибыль.

## Цели  
- Исследовать поведение пользователей разных тарифов  
- Проанализировать особенности поездок в разных городах  
- Проверить статистические гипотезы, влияющие на доход и удержание клиентов  

## Используемые технологии и библиотеки

- **Python:**  
  `pandas`, `numpy` — работа с данными и их обработка  
- **Визуализация:**  
  `matplotlib` — построение графиков и визуальный анализ  
- **Статистика:**  
  `scipy.stats` — проверка статистических гипотез и проведение тестов
